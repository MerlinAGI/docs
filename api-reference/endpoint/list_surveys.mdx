description: "This endpoint lists all surveys."
---

### Response

Returns a list of surveys.
Each survey is an object with the following fields:

<ResponseField name="id" type="uuid">
  Permanent unique identifier for the survey.
</ResponseField>

<ResponseField name="link_id" type="string">
  The link id of the survey. This is editable in the survey settings so it might
  change.
</ResponseField>
title: last_rev.title, created_at: survey.created_at, link_id: survey.link_id, desc:
`${last_rev.title} (${reply_count} Responses)`

<ResponseField name="title" type="string">
  The title of the survey.
</ResponseField>

<ResponseField name="created_at" type="string">
  UTC timestamp of when the survey was created.
</ResponseField>

<ResponseField name="desc" type="string">
  A description of the survey. Eg. "My Survey (10 Responses)"
</ResponseField>

```bash Example Request
curl 'https://talktomerlin.com/api/public/list_surveys' \
-H 'x-id: <id_token>'
```
