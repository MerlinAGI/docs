Using Merlin's API is only possible if your account is login with Google (OAuth2). In that case, use set the `x-id` header to a valid _id_token_ from your Google account.

```bash
'x-id': 'eyJh....'
```

[Documentation for Google ID tokens](https://developers.google.com/identity/openid-connect/openid-connect)
